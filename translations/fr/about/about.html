Qu'est-ce que ce logiciel

Ceci est un logiciel pour le téléchargement sécurisé de fichiers. Il crypte et décrypte les fichiers entièrement dans votre navigateur sans révéler leur contenu au serveur. Tous les fichiers uploadé expirent après une durée paramétrable pour garantir que vos informations ne sont pas disponibles sur internet pour toujours. La cryptographie peut aussi être effectué par le serveur pendant le téléchargement si le JavaScript n'est pas actif dans votre navigateur, mais cela est moins sécurisé que y effectuer dans votre navigateur.

Ce que ce logiciel n’est pas

Ceci n'est pas un logiciel magique qui garantira immédiatement la sécurité totale de les fichiers envoyés en l’utilisant. Le logiciel est conçu pour protéger vos fichiers pendant qu'il les stocke, mais il n'a aucun pouvoir concernant vos fichiers lorsqu'ils sont téléchargés. Si vous partagez un upload avec quelq'un, il n'y a rien qui lui empèche de l'envoyer ailleurs contre votre volonté. En plus, c'est possible q'un acteur malveillant pourrait compromettre ce logiciel pour voler des informations. En résumé: si vous voulez la meilleure sécurité possible, vous ne pouvez faire confiance a aucun autre. Quiconque vous dit le contraire a tort (intentionnellement ou non).
