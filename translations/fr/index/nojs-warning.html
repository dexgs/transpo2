JavaScript n'est pas actif dans votre navigateur. Par conséquent, le cryptage de vos fichiers sera effectuée par le serveur au lieu de votre navigateur. La clé de décryptage ne sera gardé par le serveur que pendant la dureé du téléchargement. Néanmoins, cela est moins sécurisé q'un téléchargement pour qui le cryptage est effectuée par votre navigateur (ceci n'est pas possible sans JavaScript).
