Was kann diese Software?

Transpo ist Software zum sicheren Austausch von Dateien über das Internet. Sie unterstützt das Verschlüsseln und Entschlüsseln von Dateien im Browser, so dass deren Inhalt nie vom Server gesehen werden kann. Alle hochgeladenen Dateien verfallen nach einer konfigurierbaren Zeitspanne, um sicherzustellen, dass Ihre Daten nicht für immer im Internet verbleiben. Die Software unterstützt auch die Verschlüsselung und Entschlüsselung auf dem Server, wenn JavaScript nicht verfügbar ist. Die Informationen, die zum Anzeigen der Dateiinhalte erforderlich sind, werden nur für die Dauer des Verschlüsselungs-/Entschlüsselungsprozesses gespeichert, aber dies ist weniger sicher als die Verschlüsselung im Browser mit JavaScript.

Was kann diese Software nicht?

Transpo ist keine Zauberei, die sofort die perfekte Sicherheit der von Ihnen mit dieser Software geteilten Daten garantiert. Die Software ist so konzipiert, dass Ihre Uploads während des Speicherns geschützt sind, aber sie hat keine Kontrolle über die Daten, die heruntergeladen wurden. Wenn Sie einen Upload mit jemandem teilen, hindert Sie nichts daran, ihn irgendwo anders hochzuladen, auch ohne Ihre Zustimmung. Es ist auch möglich, dass ein böswilliger Akteur diese Software kompromittiert, um sensible Informationen zu stehlen. Kurz gesagt: Wenn Sie die beste mögliche Sicherheit wünschen, können Sie keinen Dritten vertrauen. Jemand der das Gegenteil behauptet, lügt. Absichtlich oder nicht.
