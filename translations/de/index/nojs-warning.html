Javascript ist in ihrem Browser nicht aktiviert Das bedeutet, dass die Verschlüsselung auf dem Server statt im Browser stattfindet. Der Schlüssel, der zum Entschlüsseln des Uploads benötigt wird, wird nur während des Verschlüsselungsprozesses auf dem Server gespeichert, aber beachten Sie, dass dies weniger sicher ist als die Verschlüsselung des Uploads im Browser, welche JavaScript erfordert.
